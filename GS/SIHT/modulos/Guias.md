
# Descripción
Proporciona una interfaz centralizada para la consulta, gestión y autorización de guías mediante cargas masivas.

- **Seguimiento:** Consulta y filtrado avanzado de guías existentes (DHL, Estafeta)
- **Generación:** Importación de archivos Excel para la creación de múltiples guias.
- **Gestión de solicitudes:** Flujo de aprobación/rechazo de envíos por parte administradores.
